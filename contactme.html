How to create a Contact us!

There are multiple ways you can create a contact us using html. I recommend using a seperate .html file located within your website to bring the user to another page. In order to do this you must create a new file within your websites folder. Once you have done that you must now use your html Boilerplate and start by making a h1 heading for contact us! Now in the body you must make a paragraph for each contact information. Here is an example:
