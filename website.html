How to create your own website!

First you must download any code editor my favorite is called Atom. After downloading a code editor I recommend heading over to w3schools HTML Intro this is a really good resource to read what HTML is and how it works. Once you have an understanding of HTML you are ready to start coding your own website! To begin you must first know what topic your website will be about. This will make it a lot more easier to create content for your website. The last thing you would want is to have no idea on what to write, but it happens all the time as a coder you will get stuck from time to time!

Once you have a topic in mind we are gonna start with the HTML "Boilerplate" code. Now inside the title you will want to put what your topic is about. Next we will start adding the H1 Heading to the body code it can be anything related to your topic! Optional:You can also add a horizontal line after your heading.
